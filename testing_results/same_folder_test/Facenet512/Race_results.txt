Complete same_folder test on rfw dataset. Each test is comparing two photos of the same individual.
The test compares every possible combination of images in the same folder.
 
For 3 images in a folder the following 3 tests would be performed:
img1 - img2, img1 - img3, img 2-3
For 4 images in a folder, a total of 6 combinations are tested.

Model: Facenet512
Detector: mtcnn
